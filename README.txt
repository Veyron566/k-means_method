# k-means_method
Реализует метод к-средних
Входные данные для метода содержатся в файле .csv (их можно создавать в Excel или LibreOffice_calc). 
Пример файла:

,x1,x2,x3,x4,x5
Australia,17800,15,8,7.3,1.9
Austria,8000,12,11,6.7,1.5
Argentina,33900,20,9,25.6,2.8
Bangladesh,125000,35,11,106,4.7

Здесь первая строка - это имена переменных, первый столбец: имена объектов.
Очевидно, что количество переменных должно быть одинаково для всех элементов. Первая клетка (первый столбец и первая строка) оставляется пустым. 
Для запуска программы запускается файл window.py. В файле logic.py содержится метод k - средних и некоторые дополнительные функции. 
Окно программы представляет из себя фрейм с панелью фиксированного размера. Панель использует координатор типа BoxSizer для координирования виджетов. Реализован класс Validator который отвечает за то, чтобы пользователь не мог вводить количество классов с помощью букв (обрабатывается только Backspace и Tab). 
Программа позволяет строить двумерный график с выбором двух переменных.
Реализована возможность созранять результаты в файл и выбирать файл с данными с помощью инстанса класса FileDialog.
